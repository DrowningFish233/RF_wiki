理智系统

理智系统是本整合包的核心机制之一，它会根据玩家的行为和游戏环境动态变化。

理智值分为三个等级：正常、警告和危险，每个等级都会影响游戏的不同方面。

理智值扩展

理智值不仅影响视觉效果，还会影响游戏机制，包括但不限于：

  • 怪物生成和强度
  • 物品掉落率
  • 特殊事件触发
  • 装备属性变化

设定解释

整合包的世界观设定融合了克苏鲁神话元素和黑暗奇幻风格，理智系统正是这一设定的体现。

傲慢

傲慢会影响玩家的攻击力和防御力，但会降低与其他玩家的互动效果。

嫉妒

嫉妒会使玩家对稀有物品有更强的渴望，同时影响战利品掉落。
