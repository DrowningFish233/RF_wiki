理智值系统

理智值是衡量角色精神状态的重要指标，会随着游戏中的各种事件而波动。高理智值提供增益效果，而低理智值则会导致罪孽侵蚀与部分负面效果。

理智值恢复

当生命值大于95%且饱食度>18时,缓慢恢复理智值。

理智值消耗

当生命值小于40%,缓慢降低理智值；当生命值小于20%,理智值降低速率翻倍。

词缀影响

拥有部分特殊饰品时,玩家的理智值会因为饰品的特殊效果,发挥出更大的作用

自定义装备系统

通过寂静装备，玩家可以自定义装备的属性和特殊效果。

材料分配

使用独特的金属来强化装备的各个方面。每种材料提供不同的属性加成：

  • 神圣金锭：镰刀熟练度+15% 神圣法术强度+10%
  • 魂染钢锭：镰刀熟练度+25% 使你额外获得10点的灵魂护盾,灵魂护盾恢复速率+10% 收集到精魂时所触发效果的额外强度+10%
  • 灌注金锭：根据经验等级提高攻击力,每30级提高1点攻击力,上限900级

特殊效果

为装备添加独特的被动能力和主动技能：

  • 持有凋零合金锭词缀效果时,玩家每20秒会获得一发虎标弹,上限8发,可使用武器技能按键消耗1枚虎标弹来推进自身,并给予自身速度2 攻击会消耗1发虎标弹,造成大额击退并额外造成20%伤害(独立乘区)
  • 受到致死伤害后，清除所有经验，并使使用者获得相当于[经验等级/10]的伤害吸收效果，持续时间为[经验等级]秒，冷却时间600秒
  • 降低玩家移动速度，受击目标会受到反向击退以靠近玩家
