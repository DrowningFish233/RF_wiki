理智值系统

理智值是衡量角色精神状态的重要指标，会随着游戏中的各种事件而波动。高理智值提供增益效果，而低理智值则会导致罪孽侵蚀与部分负面效果。

理智值恢复

当生命值大于95%且饱食度>18时,缓慢恢复理智值。

理智值消耗

当生命值小于40%,缓慢降低理智值 当生命值小于20%,理智值降低速率在小于40%的基础上翻倍
